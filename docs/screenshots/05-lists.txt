D. SIDEBAR (left rail)
------------------------------------------------------------
Quick Access:
  Home (active) — user lists + smart lists
  All — every list in the app

Folders:
  Next Actions — scheduling-oriented lists + smart daily/weekly/monthly buckets
  Scheduled — time-bucket lists synced from scheduler
  + New Folder

Each folder can contain lists and an auto-managed **All Items** pool
(__all-items__{folderId}) for uncategorized items.

Orb gallery (1040+ PNGs from orbs-manifest) + user upload with bg removal.
