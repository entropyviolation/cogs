------------------------------------------------------------
A. GLOBAL HEADER (visible on every screen)
------------------------------------------------------------
Rendered by app/page.tsx. "COGS" title on the left; action controls on the right.

1. "Review" button  [ClipboardCheck + badge]  (components/Reviews/reviews.tsx)
   - Dropdown of day/week/month/quarter/year reviews.
   - Badge shows how many just-ended periods have not been reviewed yet (5 here).
   - Opens the Review dialog: carry-over unfinished tasks, gratitude, reflection
     questions, saved plan text, and plan reflection field.

2. "Tracking" button  [Brain icon]  (components/cognitive-state.tsx)
   - Opens a dialog with the compact TimeGrid (scope pens + minute grid).
   - Replaces the old slider-based cognitive-state form.

3. "Inbox" button  [Inbox icon + count badge]  (components/inbox.tsx)
   - Badge = unprocessed inbox tasks (category === "inbox", not completed). Here: 1.
   - Opens the Inbox dialog with Clarify / Delete per task and "Clarify All".

4. "Bulk Add" button  (components/enhanced-bulk-add.tsx)
   - Multi-line capture; "Category:" headers create/assign categories.

5. "Quick Add" button  [Plus, primary]  (components/quick-add.tsx)
   - Single-field capture straight to the inbox.

------------------------------------------------------------

------------------------------------------------------------
1. Date card (left)

   - Contains the Points Stats grid (section D).

2. "Today's Progress" card (right)  (daily-progress-quickview.tsx)

------------------------------------------------------------
F. HABIT TRACKER (components/Home/Habits/habit-tracker.tsx)
------------------------------------------------------------
State lives in lib/habits-store.ts (key cogs-habits-store), shared with Lists
Daily Habits view.

1. Frequency tabs: "Daily (14)" | "Weekly (0)" | "Monthly (0)"
   - Daily shows the week grid (this screenshot).
   - Weekly / Monthly show PeriodHabitList cards instead.

2. Week navigation pill (week-navigation.tsx) — Daily tab only
   - "<" previous week, "Today", date range (e.g. "Jun 15 – Jun 21, 2026"), ">".

3. "Hide completed today" toggle — filters rows already done for today.

4. "Settings" button — habits import/export/reset dialog.

5. Floating "+" button — opens Task Form dialog to add/edit a habit.

------------------------------------------------------------
G. HABIT TABLE / TASK GRID (task-grid.tsx)
------------------------------------------------------------
Spreadsheet table. Columns: Task | Mon..Sun (today highlighted) | Completion | Actions.

Habit input types (five types):
  BOOLEAN  — checkbox per day (Drink water, Stretch, …)
  TIME     — minutes input / goal (Work 1h, Exercise 30m, Clean 15m)
  COUNT    — numeric / goal (Write 3 pages, Read 10 pages)
  INCREMENTAL — multi-metric with rising daily goals (Chess/puzzle, Meditation)
  TEXT     — textarea per day (Do something artistic/creative)

Per-row: weekly completion bar + %; Edit / Delete actions.
Bottom row: "Daily Completion" — aggregate % per weekday column.
Completing a habit awards points via points-store.
