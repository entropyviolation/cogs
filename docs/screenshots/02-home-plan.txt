------------------------------------------------------------
Left rail title varies by view — here "Planned This Month" with count badge (0).
Lists tasks scheduled for the current period but not yet placed on the calendar.
Empty state: "Nothing to plan here." Items are draggable onto calendar cells;
clicking opens task-detail-popup.

D. MONTH VIEW CALENDAR (month-view.tsx)
------------------------------------------------------------
1. Navigation: "<" | "June 2026" | ">" | "Today".
2. 7-column grid (Sun–Sat), 6 rows. Today (Jun 20) highlighted with gradient +
   pulsing icon. Adjacent-month days dimmed.
3. Event chips on days (colored by event): Jun 20 "02:00 Product Strat…",
   Jun 22 "01:00 Kayaking W…". Task chips appear sage-green with Clock icon.
4. Drag-drop: tasks onto days set scheduledDate; events move by date.
5. Click empty day → Add Event prefilled; click event/task → edit/detail.
