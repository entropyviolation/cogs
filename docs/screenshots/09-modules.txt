------------------------------------------------------------
B. MODULES PANEL HEADER (modules-panel.tsx)
------------------------------------------------------------
Title "Modules" — "Composable views built from your lists, tasks, and stats."
"+ Add Module" button — pick module type, title, and configuration.

Modules persist in lib/modules-store.ts (key cogs-modules-store).
Each card has Settings (gear) and Remove (X) in the header.

------------------------------------------------------------
C. MODULE TYPES SHOWN
------------------------------------------------------------
1. "Points this week"  [BarChart3 icon]  — analytics-stat module
   - Headline number from points-store for the current week (0 here).

2. "Writing Assignment Generator"  [PenLine icon]  — writing-prompt module
   - Random creative prompt (e.g. "Write an open letter about the city at 3am,
     in under 300 words.").
   - "New prompt" button rerolls.

3. "What should I do now?"  [Shuffle icon]  — random-task module
   - Surfaces an open task ("Example task", 60m, importance 5).
   - "Another" rerolls; "Mark done" completes via task-store.

------------------------------------------------------------
D. OTHER MODULE TYPES (available via Add Module)
------------------------------------------------------------
  list-explorer  — random item from a configured list (reading list, cleaning, …)
  list-summary   — progress stats on a chosen list
  analytics-stat — configurable headline metric (points, tasks, habits, …)

Modules can filter list items by attribute rules configured in Settings.
